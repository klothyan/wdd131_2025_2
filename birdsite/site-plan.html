Birdsite Site Plan

Kyleigh Lothyan

WDD 131

Overview

Purpose

Birdsite will be a website that allows its users to view and learn about the birds of southeastern Idaho. It will be informative in providing sounds that various common birds make as well as including images/facts about each one.

Audience

My audience is anyone who wishes to learn about birds for fun, or anyone who is interested in the behavior or interaction of birds within Rexburg/southerneastern Idaho.

Dynamic elements

For the homepage, I will use JavaScript to make a search bar. This will allow users to be able to search for certain birds. On the second page, I'll use JavaScript to allow users to be able to hover over an image of a bird and hear the sounds that it makes.
